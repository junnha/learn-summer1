# 0x03강에서의 배열 활용 개별정리
-------------
### 반복 입력/저장

- 여러 개의 값을 한 번에 저장해서 반복 처리할 때 `(ex. for(int i=0;i<n;i++) cin >> arr[i];)`

---

### 카운팅/분류

- 숫자를 자릿수별로 확인하여 배열에 저장하는 방법
```cpp
while(N){
    a[N%10]++;
    N /= 10;
  }
```
### 그룹별 개수 누적 및 그룹 처리

- 2차원 배열 등으로 그룹별로 값을 누적
- 그룹마다 반복문 돌며 조건에 맞게 처리 (ex. 방 배정, 분반 등)

```cpp
int s[2][7] = {};
for (int i=0; i<N; ++i) {
  int a, b;
  cin >> a >> b;
  s[a][b]++;
}
```
