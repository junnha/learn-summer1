## 스택 (Stack)

---

### 스택의 성질
- 원소의 추가 O(1)
- 원소의 제거 O(1)
- 제일 상단의 원소 확인이 O(1)
- 제일 상단이 아닌 원소의 확인/변경 불가능

### 기능과 구현

```cpp
const int MX = 1000005;
int dat[MX];
int pos = 0;

void push(int x){
    dat[pos++]=x;
}

void pop(){
    pos--;
}

int top(){
    return dat[pos-1];
}
```
---
#### push
- 스택의 맨 위에 요소를 추가
- `s.push(10);`

#### pop
- 스택의 맨 위 요소를 제거
- `s.pop();`

#### top
- 스택의 맨 위 요소에 접근 (읽기/수정 가능)
- `int x = s.top();`

#### empty
- 스택이 비어 있는지 확인
- `if (s.empty()) { ... }`

#### size
- 스택에 들어 있는 요소 개수 반환
- `int n = s.size();`
